# Tucil3_13520069

15 Puzzle Solver

## Description

Using **branch and bound** algorithm to solve 15 puzzle.

## Getting Started

### Prerequisite

- Python 3
- Tkinter (GUI)

### Installation

1. Clone repo

   ```sh
   git clone https://github.com/kentlius/Tucil3_13520069.git
   ```

2. Changedir to repo source
   ```sh
   cd Tucil3_13520069/src
   ```

## Usage

- For CLI

  ```sh
  python main.py
  ```

- For GUI
  ```sh
  python gui.py
  ```
